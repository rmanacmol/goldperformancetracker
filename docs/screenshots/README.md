# Demo Screenshots

Add PNG captures here before submission. Suggested filenames:

| File | What to capture |
|------|-----------------|
| `01_player_list.png` | Player list (light theme), search + club chips visible |
| `02_player_detail.png` | Player detail with stats card and shot list |
| `03_stats_chart.png` | Stats screen — bar chart + carry progress |
| `04_offline_mode.png` | Airplane mode on — offline banner + cached list |
| `05_dark_theme.png` | Any screen in dark theme (optional) |

## How to capture

1. Run the app on an emulator (Pixel 6, API 34+ recommended).
2. Android Studio → **View → Tool Windows → Logcat** (optional, for verifying sync logs).
3. Use emulator **Camera** icon or `Cmd+S` / **Extended Controls → Screenshots**.
4. Save files into this folder and reference them from `ARCHITECTURE.md`.
